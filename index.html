📖 Item Naming Guide — Noun first, descriptor second

Filing items by their main noun keeps your list grouped at the store. Use Noun, Descriptor format for the Sort Name so similar items stay together alphabetically.

✓ Pepper, Bell - Red✗ Red Bell Pepper
✓ Pepper, Bell - Green✗ Green Pepper
✓ Carrot, Baby✗ Baby Carrots
✓ Carrot, Shredded✗ Shredded Carrots
✓ Tomato, Cherry✗ Cherry Tomatoes
✓ Potato, Russet✗ Russet Potato
✓ Mushroom, Sliced✗ Sliced Mushrooms
✓ Cheese, Cheddar✗ Cheddar Cheese
✓ Butter, Salted✗ Salted Butter
✓ Juice, Apple✗ Apple Juice
✓ Chicken, Breast✗ Chicken Breast
✓ Sausage, Italian✗ Italian Sausage

The Display Name can be written naturally (e.g. "Red Bell Pepper") — that's what appears on your shopping list. The Sort Name is just for filing order.
